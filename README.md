# 🎮 Unreal PRO Visual Node Editor (Qt C++)

A modern **Unreal-style Visual Scripting Editor** built using **Qt6 C++ GraphicsView Framework**.  
This project demonstrates a fully functional node-based execution engine with animated execution flow, logical operations, and a customizable visual graph interface.

Designed as a learning and portfolio project inspired by **Unreal Engine Blueprints** and modern visual programming tools.

---

## ✨ Features

### 🧩 Visual Node System
- Drag & drop node creation
- Neon-styled UI with glow effects
- Dynamic wire connections
- Zoom & pan graph canvas

### ⚡ Execution Engine
- Multi-branch dataflow execution
- Queue-based Unreal-style processing
- Logical & arithmetic node support
- Real-time output display

### 🔀 Supported Nodes
- Variable
- Add / Subtract / Multiply / Divide
- Logical AND / OR / NOT
- Compare (Unreal-style operator selector)
- If branching (Yes / No pins)
- Output display node

### 🎬 Unreal-Style Visual Effects
- Animated execution flow
- Neon spline wires
- Active node highlighting
- Smooth execution traversal

---

## 🏗️ Tech Stack

- **Language:** C++
- **Framework:** Qt 6
- **Rendering:** QGraphicsView Framework
- **Architecture:** Node Graph + Execution Engine
- **Build System:** CMake
